아파트를 사기로 결정하면 곧바로 따라오는 질문이 있습니다. "남편 단독명의로 할까, 부부 공동명의로 할까?"

주변에서 "공동명의 하면 세금이 줄어든다"는 이야기를 많이 들으시지만, 구체적으로 어디서 얼마나 줄어드는지는 잘 모르시는 경우가 많습니다. 이 글에서는 아파트를 처음 살 때 공동명의로 하면 세금이 어떻게 달라지는지, 시가 구간별로 정리합니다.

## 1. 공동명의로 하면 취득세가 더 나올까

아닙니다. 취득세는 주택 전체 가액을 기준으로 세율이 정해지고, 공동명의라면 지분에 따라 나눠서 낼 뿐입니다. 단독명의로 사든 공동명의로 사든 취득세 총액은 같습니다. 공동명의를 한다고 해서 취득세가 추가로 발생하지 않습니다.

이것이 "처음 살 때 공동명의"가 중요한 이유입니다. 나중에 단독명의를 공동명의로 바꾸려면 배우자에게 지분을 증여해야 하고, 그 과정에서 취득세(3.5%)와 증여세가 별도로 발생합니다. 처음부터 공동명의로 하면 이 비용이 전혀 들지 않습니다.

## 2. 종합부동산세에서 가장 큰 차이가 납니다

종합부동산세는 사람별로 과세됩니다. 단독명의면 한 사람의 공시가격으로 세금이 계산되고, 공동명의면 공시가격이 절반씩 나뉘어 각각 계산됩니다.

공제 금액도 다릅니다. 1세대 1주택 단독명의는 12억원이 공제되고, 공동명의는 각각 9억원씩 합계 18억원이 공제됩니다. 즉, 공동명의가 공제 범위가 더 넓습니다.

공시가격 구간별로 종부세가 어떻게 달라지는지 비교하겠습니다.

| 공시가격 | 단독명의 종부세 (연) | 공동명의 종부세 (연) |
|---|---|---|
| 10억원 | 0원 (12억 공제 내) | 0원 (각 5억 < 9억 공제) |
| 14억원 | 약 60만원 | 0원 (각 7억 < 9억 공제) |
| 18억원 | 약 180만원 | 0원 (각 9억 = 9억 공제) |
| 21억원 | 약 340만원 | 약 90만원 |
| 25억원 | 약 560만원 | 약 200만원 |

공시가격 18억원까지는 공동명의일 때 종부세가 아예 나오지 않습니다. 단독명의에서는 12억원을 초과하는 순간부터 세금이 나오기 시작합니다.

공시가격이 18억원을 넘으면 공동명의에서도 종부세가 발생하지만, 과세표준이 절반으로 나뉘기 때문에 낮은 세율 구간이 적용되어 세액이 줄어듭니다.

### 고령자·장기보유 세액공제는 어떻게 되나

"공동명의 하면 고령자·장기보유 세액공제를 못 받는다"는 말을 들으신 적이 있을 수 있습니다. 예전에는 그랬지만, 2021년에 법이 바뀌었습니다.

종합부동산세법 제10조의2에 따라 부부 공동명의 1주택자는 매년 신고할 때 두 가지 방식 중 하나를 고를 수 있습니다.

| 방식 | 공제 | 고령자·장기보유 세액공제 |
|---|---|---|
| 인별 과세 | 각 9억원 (합계 18억원) | 받을 수 없음 |
| 1세대1주택 특례 신청 | 12억원 | 받을 수 있음 (최대 80% 감면) |

젊을 때는 인별 과세를 선택해서 넓은 공제를 받다가, 60세가 넘어 고령자 세액공제가 유리해지면 특례로 전환할 수 있습니다. 이 선택은 매년 바꿀 수 있으므로, 공동명의라고 해서 포기해야 하는 것은 없습니다.

고령자·장기보유 세액공제율은 다음과 같습니다.

| 구분 | 요건 | 공제율 |
|---|---|---|
| 고령자 | 만 60~64세 | 20% |
| | 만 65~69세 | 30% |
| | 만 70세 이상 | 40% |
| 장기보유 | 5~9년 | 20% |
| | 10~14년 | 40% |
| | 15년 이상 | 50% |
| 합산 한도 | | 최대 80% |

## 3. 양도소득세에서도 차이가 있습니다

1세대 1주택으로서 실거래가 12억원 이하이면 양도소득세가 비과세됩니다. 이 비과세는 세대 기준이므로 단독명의든 공동명의든 동일하게 적용됩니다. 12억원 이하 아파트라면 명의 구분이 양도세에 영향을 주지 않습니다.

12억원을 초과하는 고가 아파트를 파는 경우에 차이가 생깁니다. 양도소득세는 양도 차익에 누진세율(6%~45%)이 적용되는데, 공동명의면 차익이 부부에게 절반씩 나뉘어 각각 낮은 세율 구간이 적용됩니다.

예를 들어 양도 차익이 10억원이라면, 단독명의에서는 10억원 전체가 한 사람에게 잡혀 42% 구간까지 올라갑니다. 공동명의면 각 5억원으로 나뉘어 40% 구간에 머뭅니다. 차익이 클수록 이 세율 차이에서 오는 세액 차이도 커집니다.

장기보유특별공제(보유·거주 기간에 따라 최대 80% 공제)는 단독명의와 공동명의 모두 동일하게 적용됩니다.

## 4. 재산세는 차이가 없습니다

재산세는 주택 자체에 부과되는 세금이므로, 공동명의로 해도 세액 총합은 달라지지 않습니다. 지분에 따라 나눠서 낼 뿐입니다.

## 5. 정리

| 세목 | 공동명의와 단독명의의 차이 |
|---|---|
| 취득세 | 최초 취득 시 차이 없음 |
| 종합부동산세 | 공동명의 시 공제 범위가 넓고, 세율 분산 효과. 고령자·장기보유 공제도 특례 신청으로 가능 |
| 양도소득세 | 12억 이하 비과세는 동일. 12억 초과 시 차익 분산으로 세율 하향 가능 |
| 재산세 | 차이 없음 |

최초 취득 시 공동명의는 추가 비용이 없고, 종부세와 양도세에서 구조적으로 유리한 위치를 가져갈 수 있습니다.

## 6. 확인 사항

- [ ] 아파트 공시가격이 12억원(단독 공제)을 넘는지 확인했습니까
- [ ] 공시가격이 18억원(공동명의 인별 공제 합계) 이하인지 확인했습니까
- [ ] 12억원 초과 고가주택이라면 양도 시 차익 분산 효과를 검토했습니까
- [ ] 향후 고령자·장기보유 세액공제가 필요한 시점에 특례 전환이 가능함을 알고 있습니까
